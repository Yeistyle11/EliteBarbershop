# Documentación del Proyecto Barbershop

Este directorio contiene archivos base del proyecto. 

## Próximos Pasos

1. Instalar dependencias: `pnpm install`
2. Configurar PostgreSQL y actualizar `.env`
3. Ejecutar migraciones: `pnpm db:push`
4. Poblar base de datos: `pnpm db:seed`
5. Iniciar desarrollo: `pnpm dev`

Revisa el README.md principal para más información.
